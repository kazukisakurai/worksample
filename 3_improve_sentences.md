`質問`
```
「selfについて」説明文が難しかったです。
```
`該当文`

```
selfはインスタンスオブジェクト自身を指しています。クラスの中でインスタンスオブジェクトを呼び出す際やそのインスタンスオブジェクトを呼び出す際は、selfを使用します。
```

`修正後`

```
selfは、「インスタンスメソッドの中で、インスタンス変数を扱う」場合 や、「class内で、そのclass自体に働きかけるアクションを作成する」場合（クラスメソッド）などに使用します。

クラスの定義中、インスタンスメソッドを作成している時点では、まだインスタンスそのものは作成されていません。これでは、複数のインスタンスごとに変数を呼び出すメソッドを作ることができません。

そこで、インスタンスメソッド内では、インスタンス名の代わりに self を指定しておくことで、インスタンスが作成された後は、インスタンス自身の名前が self に代入されることになり、インスタンス変数を使用したアクションを実行することができます。
```
