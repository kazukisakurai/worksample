`質問`
```
「selfについて」説明文が難しかったです。
```
`該当文`

```
selfはインスタンスオブジェクト自身を指しています。クラスの中でインスタンスオブジェクトを呼び出す際やそのインスタンスオブジェクトを呼び出す際は、selfを使用します。
```

`修正後`

```
selfは、「インスタンスメソッドの中で、インスタンス変数(←ここに使用)を扱う」場合に使用します。

クラスの定義中、インスタンスメソッドを作成している時点では、まだインスタンスそのものは作成されていません。これでは、インスタンスごとに変数を呼び出すメソッドを作ることができません。

そこで、インスタンスが作成された後、あらかじめ定義してあるインスタンスメソッドの中でインスタンス変数を扱えるよう、インスタンス自身の名前を代入するために、selfを使用します。
```
